Detailed Description

A diagnostic message bundled with the source location where it was written.

The core helpers (raise(), check(), throw_if(), check_not_null()) take a Diagnostic in place of an explicit source location. Its file/line/function arguments default to the compiler builtins __builtin_FILE()/__builtin_LINE()/ __builtin_FUNCTION(), which capture the caller's location, so a string literal or std::string passed at the call site is implicitly wrapped into a Diagnostic that records exactly where the call appears – callers pass no explicit location:

svmp::check<MyException>(ptr != nullptr, "pointer must not be null");
