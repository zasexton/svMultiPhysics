Reference Lagrange nodes with their integer lattice indices.

Returns the same coordinates as get_lagrange_node_coords(), paired with the integer lattice index of each node (see LagrangeNodeLayout). The structural invariants in the contract (size match, components in [0, order], simplex/wedge sum bounds) are validated before returning.

Reference nodes for an arbitrary-order serendipity layout.

Generates the stratified serendipity node set for the quadrilateral or hexahedral family at the requested order: the corner+edge skeleton (the leading prefix of the complete Lagrange layout of the same order, in VTK boundary order) followed by the reduced face and volume interior. This is the single source of serendipity node geometry – SerendipityBasis builds its mode space and coefficient table on top of these coordinates for both the arbitrary-order path and the named Quad8/Hex20 layouts (the order-2 instances). Wedge serendipity (Wedge15) is a fixed named layout and is not generated here.

Parameters
topologyBasisTopology::Quadrilateral or BasisTopology::Hexahedron.
orderPolynomial order; must be >= 1.
Returns
Reference node coordinates in stratified (skeleton-then-interior) order.
Exceptions
BasisConstructionExceptionIf order is below 1.
BasisElementCompatibilityExceptionIf topology is not Quadrilateral or Hexahedron.
