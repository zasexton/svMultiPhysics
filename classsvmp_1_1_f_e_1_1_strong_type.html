Detailed Description

template<typename T, typename Tag>
class svmp::FE::StrongType< T, Tag >

Strong type wrapper template for type-safe programming.

Prevents accidental mixing of conceptually different types that have the same underlying representation.

Template Parameters
TUnderlying value type
TagEmpty tag type that distinguishes otherwise identical wrappers
