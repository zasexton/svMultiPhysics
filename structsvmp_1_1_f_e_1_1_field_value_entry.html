Detailed Description

Field value at an evaluation point — scalar, vector, or tensor.

Used by PointEvaluator and the auxiliary assembly path to supply FE field values at entity locations (e.g., nodal DOF values for Node-scoped auxiliary models with Lagrange Kronecker delta).
