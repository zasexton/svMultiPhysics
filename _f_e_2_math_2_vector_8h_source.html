7/**
8 * @file Vector.h
9 * @brief Fixed-size vector types for FE computations, backed by Eigen.
10 *
11 * The FE library standardizes on Eigen for linear algebra. These aliases give
12 * element-level code a stable vocabulary type without re-exporting all of
13 * Eigen. Note that, unlike the previous in-house implementation, Eigen types
14 * are NOT zero-initialized by default construction; use Vector::Zero() where a
15 * zeroed value is required.
16 *
17 * This is a small, fixed-size (compile-time length) vector for element-level FE
18 * kernels in namespace svmp::FE::math. It is distinct from, and not a replacement
19 * for, the legacy dynamically sized container in solver/Vector.h: the two differ
20 * in namespace, size model (compile-time vs runtime), and memory management, and
21 * coexist deliberately.
22 */
23
